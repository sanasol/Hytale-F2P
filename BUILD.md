# Build Instructions

## Prerequisites

Install dependencies:
```bash
npm install
```

## Building

### Build for current platform:
```bash
npm run build
```

### Build for specific platform:

**Windows:**
```bash
npm run build:win
```

**Linux:**
```bash
npm run build:linux
```

**macOS:**
```bash
npm run build:mac
```

### Build for all platforms:
```bash
npm run build:all
```

Built executables will be in the `dist/` directory
